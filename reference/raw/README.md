# Raw source layer

`raw/`는 공식 원자료를 가리키는 날짜별 증거 레이어다. 저작권이 있는 외부 문서를 통째로 복제하지 않고 다음만 기록한다.

- 원문 URL
- 확인일
- 해당 자료가 뒷받침하는 핵심 사실
- 소스 코드라면 가능한 한 고정 커밋
- 자료 간 모순 또는 주의점

과거 스냅샷은 수정하지 않는다. 새로운 조사 결과는 새 날짜 파일로 추가하고 `reference/wiki/`의 현재 문서를 갱신한다.
